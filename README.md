# Turista Facoltoso - Backend

Backend del progetto "Turista Facoltoso" sviluppato in Java con Maven e Javalin.

## Stato
- Inizializzazione progetto Maven
- Configurazione Javalin
- Server backend avviabile come fat jar

## Tecnologie
- Java
- Maven
- Javalin
- PostgreSQL

### Endpoint principali

- GET /abitazioni/host/{codiceHost}
- GET /prenotazioni/ultima/{utenteId}
- GET /host/top-ultimo-mese
- GET /host/super-host
- GET /utenti/top-5-giorni-ultimo-mese

Le API sono state testate tramite Postman.
